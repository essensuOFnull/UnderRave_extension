⚙️ Настройки контекстного меню

Блокирует попытки сайтов отключить контекстное меню (например, в видеоплеерах). Стандартное меню будет появляться всегда.
Добавляет в контекстное меню пункт для запуска PiP для ближайшего видео. Работает на любых сайтах, даже если видео спрятано в теневом DOM.
ℹ️ Примечание: Изменения вступают сразу после переключения чекбоксов. Для применения на уже открытых страницах может потребоваться их перезагрузка.
